# Workflow Ideas — 도구 조합 워크플로우 아이디어 정리

> 최종 업데이트: 2026-02-09
> 기존 PPT 생성 워크플로우를 확장하여, 현재 도구 생태계로 구현 가능한 시나리오를 정리한 문서입니다.

---

## 현재 도구 생태계

| 카테고리 | 도구 | 주요 기능 |
|---------|------|----------|
| 검색 | `google_search` | 웹/이미지/영상 검색, 위치/언어 필터 |
| 검색 | `web_search` (Exa) | 학술/금융/GitHub/LinkedIn 특화 검색 |
| 검색 | `twitter_search` | X/Twitter 고급 검색 |
| 검색 | `youtube_search` | YouTube 영상 검색 |
| 검색 | `link_reader` | URL 콘텐츠 추출 |
| 검색 | `youtube_link_analyzer` | YouTube 영상 정보 + 자막 추출 |
| 이미지 | `gemini_image_tool` | 4K 이미지 생성/편집, 14장 입력, 다국어 텍스트 |
| 이미지 | `seedream_image_tool` | 4K 시네마틱 이미지 생성, 10장 입력 |
| 영상 | `wan25_video` | text→video, image→video (5~10초) |
| 영상 | `grok_video` | text→video, image→video, video edit (1~15초) |
| 코드 | `run_python_code` | Python 샌드박스 (pandas, matplotlib, numpy, pip install) |
| 파일 | `read_file`, `write_file`, `apply_edits`, `delete_file`, `grep_file` | 워크스페이스 파일 CRUD |
| 유틸 | `calculator`, `get_file_info`, `list_workspace` | 계산, 파일 메타데이터 |

---

## 핵심 패턴

```
리서치/입력 → 시각 에셋 생성 → 코드로 조립/패키징 → 파일 전달
```

## 구현 예정 (우선순위 순)

### 1. PDF 보고서 생성 (Research Report)

- **상태**: 🟢 구현 완료
- **도구 체인**: `google_search`/`web_search` → `run_python_code` (데이터 분석 + matplotlib 차트) → `gemini_image_tool` (표지/헤더 이미지) → `run_python_code` (reportlab으로 PDF 조립)
- **사용 시나리오**:
  - "AI 시장 분석 보고서 만들어줘"
  - "2025년 반도체 산업 트렌드 리포트 PDF로 만들어줘"
  - "우리 회사 매출 데이터 분석 보고서 작성해줘" (CSV 업로드)
- **테스트 인풋**:
  - `2026년 글로벌 AI 시장 현황과 전망을 조사해서 PDF 보고서로 만들어줘. 표지 이미지도 포함하고, 시장 규모 차트랑 주요 기업 비교표도 넣어줘.`
  - `한국 전기차 시장 분석 리포트를 PDF로 작성해줘. 최근 판매량 추이 그래프, 주요 제조사 시장 점유율 파이차트, 정부 보조금 정책 요약을 포함해줘.`
  - `비트코인과 이더리움의 2025년 가격 동향을 분석한 PDF 보고서 만들어줘. 월별 가격 추이 차트, 주요 이벤트 타임라인, 투자 전망 섹션을 넣어줘.`
- **기술 포인트**:
  - `reportlab` pip install로 PDF 생성
  - 표지 이미지: gemini_image_tool로 4K 생성
  - 차트: matplotlib으로 생성 후 PDF에 삽입
  - 텍스트: 검색 결과 요약을 본문에 배치
- **예상 구조**:
  1. 표지 (이미지 기반)
  2. 목차
  3. 요약 (Executive Summary)
  4. 본문 섹션들 (텍스트 + 차트 + 표)
  5. 결론/권고사항
  6. 출처/참고문헌

### 2. 인포그래픽 생성 (Infographic)

- **상태**: 🟢 구현 완료
- **도구 체인**: `gemini_image_tool` (섹션별 이미지, 3:4 비율) → `run_python_code` (PIL로 세로 합성 + PDF 변환)
- **사용 시나리오**:
  - "기후변화 인포그래픽 만들어줘"
  - "우리 서비스 소개 인포그래픽 만들어줘"
  - "건강한 식습관 인포그래픽 만들어줘"
- **테스트 인풋**:
  - `커피의 역사와 종류를 정리한 인포그래픽 만들어줘. 커피 원두 종류, 추출 방법, 나라별 소비량 같은 내용을 예쁘게 정리해줘.`
  - `수면의 과학 인포그래픽 만들어줘. 수면 단계, 적정 수면 시간, 수면 부족의 영향, 숙면을 위한 팁 등을 플랫 디자인 스타일로 만들어줘.`
  - `스타트업 창업 단계를 정리한 인포그래픽 만들어줘. 아이디어 발굴부터 투자 유치까지 단계별로 아이콘과 함께 깔끔하게 정리해줘.`
- **기술 포인트**:
  - 섹션 분리: 헤더, 본문 블록 3~5개, 푸터
  - 각 섹션을 동일 스타일로 gemini_image_tool 생성
  - PIL(Pillow)로 세로 합성 → 하나의 긴 이미지
  - 선택적으로 PDF 변환
- **스타일 일관성**:
  - 첫 번째 프롬프트에서 컬러 팔레트/폰트/배경 정의
  - 이후 모든 섹션에 동일 스타일 키워드 반복

### 3. 소셜 미디어 컨텐츠 팩 (Social Media Content Pack)

- **상태**: 🟢 구현 완료
- **도구 체인**: `gemini_image_tool` (다양한 비율) → `run_python_code` (ZIP 패키징)
- **사용 시나리오**:
  - "신제품 런칭 SNS 이미지 세트 만들어줘"
  - "카페 오픈 홍보 소셜 미디어 팩 만들어줘"
  - "유튜브 채널 브랜딩 이미지 만들어줘"
- **테스트 인풋**:
  - `"FLAVOR BEAN"이라는 이름의 디저트 카페 오픈 홍보용 소셜 미디어 이미지 세트 만들어줘. 파스텔 톤 분위기로 인스타 피드, 스토리, 유튜브 썸네일, 트위터 헤더까지 전부 만들어서 ZIP으로 묶어줘.`
  - `AI 프로그래밍 강의 런칭 SNS 홍보 이미지 팩 만들어줘. "코딩의 미래, AI와 함께"라는 슬로건으로 인스타그램, 유튜브, 트위터용 이미지를 만들어줘. 다크 테마에 네온 포인트 스타일로.`
  - `여행 유튜브 채널 "세계한바퀴" 브랜딩 이미지 세트 만들어줘. 채널 배너, 썸네일 템플릿, 인스타 프로필용 이미지를 통일감 있게 만들어줘.`
- **생성 이미지 목록**:
  - Instagram 피드 (1:1) — 4K
  - Instagram/TikTok Story (9:16) — 4K
  - YouTube 썸네일 (16:9) — 4K
  - Twitter/X 헤더 (3:1 or 21:9) — 4K
  - Facebook 커버 (16:9) — 4K
  - 프로필 이미지 (1:1) — 2K
- **기술 포인트**:
  - 동일 컨셉으로 비율만 다르게 생성
  - `zipfile` 모듈로 전체 패키징
  - 파일명에 플랫폼명 포함 (예: `instagram_feed_1080x1080.png`)

### 4. 엑셀 데이터 분석 리포트 (Excel Analysis Report)

- **상태**: 🟢 구현 완료
- **도구 체인**: `run_python_code` (pandas 분석 + openpyxl 엑셀 생성 + matplotlib 차트 삽입)
- **사용 시나리오**:
  - "이 CSV 파일 분석해서 엑셀 리포트 만들어줘" (파일 업로드)
  - "매출 데이터 피벗 테이블이랑 차트 포함된 엑셀 만들어줘"
  - "설문조사 결과 분석 엑셀 리포트 만들어줘"
- **테스트 인풋**:
  - `가상의 온라인 쇼핑몰 월별 매출 데이터를 만들어서 엑셀 분석 리포트로 정리해줘. 카테고리별 매출, 월별 추이 차트, 베스트셀러 TOP 10 표, 요약 대시보드 시트를 포함해줘.`
  - `50명 학생의 국어, 영어, 수학 성적 샘플 데이터를 생성하고, 과목별 평균/표준편차, 성적 분포 히스토그램, 과목 간 상관관계 분석이 담긴 엑셀 리포트를 만들어줘.`
  - `2023년~2025년 분기별 스타트업 투자 유치 샘플 데이터를 만들어서 연도별 투자 추이 그래프, 산업별 투자 비중 파이차트, 피벗 테이블이 포함된 엑셀 파일을 만들어줘.`
- **기술 포인트**:
  - `openpyxl` pip install로 Excel 생성
  - 여러 시트: 원본 데이터, 요약 통계, 피벗 테이블, 차트
  - 조건부 서식, 셀 스타일링
  - matplotlib 차트를 이미지로 시트에 삽입
- **예상 시트 구조**:
  1. Dashboard (요약 + 차트)
  2. Raw Data (원본 데이터)
  3. Statistics (기술 통계)
  4. Pivot Tables (피벗 분석)
  5. Charts (시각화)

### 5. 스토리보드 / 만화 (Storyboard / Comic Strip)

- **상태**: 🟢 구현 완료
- **도구 체인**: `gemini_image_tool` (패널별 이미지, 1:1 또는 4:3) → `run_python_code` (PIL로 그리드 합성)
- **사용 시나리오**:
  - "4컷 만화 만들어줘"
  - "우리 제품 사용법 스토리보드 만들어줘"
  - "동화 그림책 만들어줘"
- **테스트 인풋**:
  - `개발자의 하루를 그린 4컷 만화 만들어줘. 1컷: 아침에 자신만만하게 출근, 2컷: 버그 발견, 3컷: 밤새 디버깅, 4컷: 원인이 세미콜론 하나. 만화 스타일로 재밌게 그려줘.`
  - `고양이가 주인공인 6컷 웹툰 만들어줘. 집사가 외출하면 고양이가 집에서 파티를 여는 이야기야. 귀여운 일본 만화 스타일로 만들어줘.`
  - `"달에 간 토끼"라는 제목의 어린이 동화 그림책 8페이지를 만들어줘. 수채화 스타일로, 토끼가 로켓을 타고 달에 가서 친구를 만나는 이야기야.`
- **기술 포인트**:
  - 패널 수: 4컷(2x2), 6컷(2x3), 8컷(2x4)
  - 각 패널 이미지에 말풍선/텍스트를 프롬프트에 포함
  - PIL로 그리드 레이아웃 합성 (패널 간격, 테두리 포함)
  - 선택적으로 PDF 변환 (여러 페이지 만화)
- **스타일 옵션**:
  - 만화 스타일 (셀 셰이딩, 만화 테두리)
  - 웹툰 스타일 (세로 스크롤)
  - 그림책 스타일 (수채화, 부드러운 톤)

## 추후 구현 검토 (아이디어 풀)

### 6. 영상 슬라이드쇼 (Video Slideshow)

- **도구 체인**: `gemini_image_tool` → `grok_video`/`wan25_video` (image→video)
- PPT의 자연스러운 확장. 각 슬라이드를 5~10초 영상 클립으로 변환.
- 줌인/줌아웃, 패럴랙스, 페이드 등 모션 추가.
- **제한**: 개별 클립 합치기는 현재 도구로 불가 (사용자가 직접 편집).
- **테스트 인풋**:
  - `"인공지능의 역사" 주제로 슬라이드 5장을 이미지로 만들고, 각 슬라이드를 줌인 효과가 들어간 5초짜리 영상 클립으로 변환해줘.`
  - `여행 사진 느낌의 제주도 소개 슬라이드 4장을 만들고, 각각 천천히 패닝하는 영상으로 만들어줘.`

### 7. 유튜브 썸네일 팩 (YouTube Thumbnail Pack)

- **도구 체인**: `youtube_search` (트렌드 분석) → `gemini_image_tool` (16:9, 4K)
- 하나의 주제로 A/B 테스트용 여러 버전 썸네일 생성.
- 인기 채널 썸네일 스타일 참고.
- **테스트 인풋**:
  - `"초보자를 위한 파이썬 강의" 유튜브 영상 썸네일을 A/B 테스트용으로 3가지 버전 만들어줘. 인기 코딩 유튜브 채널 스타일 참고해서 임팩트 있게 만들어줘.`
  - `먹방 유튜브 채널용 "역대급 매운 라면 도전" 썸네일 3종 만들어줘. 강렬한 빨간색 배경에 큰 텍스트로.`

### 8. 이력서/포트폴리오 (Resume / Portfolio Builder)

- **도구 체인**: `gemini_image_tool` (디자인 이력서 이미지) + `run_python_code` (python-docx/reportlab으로 편집 가능한 문서)
- 디자인된 이력서 이미지 + 편집 가능한 DOCX/PDF 동시 제공.
- **테스트 인풋**:
  - `프론트엔드 개발자 3년차 이력서를 만들어줘. 이름: 김철수, 기술 스택: React, TypeScript, Next.js. 모던하고 깔끔한 디자인으로 이미지 버전이랑 PDF 버전 둘 다 만들어줘.`
  - `UI/UX 디자이너 포트폴리오 PDF를 만들어줘. 가상의 프로젝트 3개를 넣고, 각 프로젝트마다 목업 이미지도 생성해서 포함해줘.`

### 9. 트렌드 리서치 대시보드 (Trend Dashboard)

- **도구 체인**: `google_search` + `twitter_search` → `run_python_code` (분석 + HTML 생성)
- 특정 키워드의 웹/SNS 트렌드를 수집하여 인터랙티브 HTML 대시보드 생성.
- Chart.js 또는 ECharts 포함 HTML.
- **테스트 인풋**:
  - `"생성형 AI" 키워드로 구글과 트위터에서 최신 트렌드를 조사하고, 주요 토픽, 감성 분석, 언급량 추이를 정리한 인터랙티브 HTML 대시보드를 만들어줘.`
  - `"전기차 vs 수소차" 키워드로 SNS 여론 분석 대시보드를 만들어줘. 긍정/부정 비율, 주요 논점, 관련 뉴스 타임라인을 차트로 보여줘.`

### 10. 브랜드 키트 (Brand Kit)

- **도구 체인**: `gemini_image_tool` (로고 변형 + 컬러 팔레트 + 사용 예시) → `run_python_code` (PDF 가이드라인)
- 로고, 컬러 팔레트, 타이포그래피, 사용 가이드라인을 하나의 PDF로 조립.
- **테스트 인풋**:
  - `"NovaTech"라는 AI 스타트업의 브랜드 키트를 만들어줘. 로고 3가지 변형(메인, 흑백, 아이콘), 컬러 팔레트, 명함 목업, 브랜드 가이드라인 PDF까지 만들어줘.`

## 참고사항

- 모든 워크플로우는 `toolPrompts.ts`에 프롬프트로 정의되며, `chatService.ts`에서 조건부로 시스템 프롬프트에 주입됨.
- PPT 워크플로우와 동일하게, 이미지 도구 + 코드 실행 도구가 모두 활성화된 경우에만 해당 프롬프트가 추가됨.
- 생성된 파일은 E2B 샌드박스 → Supabase Storage로 자동 동기화되어 사용자에게 다운로드 카드로 표시됨.
- `generated-images` 버킷은 public이므로, 이미지 URL을 Python 코드에서 직접 사용 가능.
